Реализованы тесты для следующих сценариев:

1. Добавление одной книги и проверка, что у нее нет жанра
2. Добавление одной и той же книги в словарь дважды
3. Проверка правильно установленного для книги жанра
4. Установка жанра для книги, которой нет в коллекции
5. Получение списка книг определённого жанра
6. Книги с возрастным рейтингом отсутствуют в списке книг для детей
7. Добавление в избранное книгу из коллекции
8. Добавление в избранное книги, которой нет в коллекции
9. Проверка удаления книги из избранного
10. Проверка добавления книг в избранное
